# EngageLearn – AI-Powered Student Engagement Prediction System

## Overview

EngageLearn is a web application that predicts student engagement using learning data such as quiz scores, video activity, and assignment completion. The goal is to help identify students who may need additional support.

## Features

- Student engagement score prediction
- AI/ML-based analysis
- Interactive dashboard
- Quiz performance tracking
- Video activity monitoring
- Learning progress visualization
- Performance analytics
- Responsive user interface

## Technologies Used

## How It Works

1. Student learning data is collected.
2. Passive learning indicators are analyzed.
3. Machine learning predicts engagement levels.
4. The dashboard displays engagement metrics and analytics.



## Installation

### Clone the repository

```bash
git clone https://github.com/saanvi-maram/engagelearn.git
```

### Navigate to the project

```bash
cd engagelearn
```

### Install dependencies

```bash
npm install
```

### Install Python packages

```bash
pip install -r requirements.txt
```

### Run the backend

```bash
python app.py
```

### Start the frontend

```bash
npm start
```
